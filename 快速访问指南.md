# 🚀 快速访问指南

## 📍 页面访问路径

### 管理员页面（需要登录）

1. **资产管理**：`http://localhost:3000/admin`
   - 管理资产和素材
   - 上传、编辑、删除资产

2. **用户管理**：`http://localhost:3000/admin/users`
   - 查看所有用户列表
   - 管理用户模式（计费模式、模型模式）
   - 跳转到用户积分管理

3. **兑换码管理**：`http://localhost:3000/admin/redeem-codes`
   - 生成兑换码
   - 查看兑换码列表和统计
   - 禁用兑换码

4. **积分管理**：`http://localhost:3000/admin/credits?userId=xxx`
   - 查看用户积分详情
   - 手动充值积分
   - 查看交易记录

### 普通用户页面

1. **梦工厂**：`http://localhost:3000/dream-factory`
   - AI 生成图片/视频
   - 查看积分余额
   - 切换模式

2. **我的积分**：`http://localhost:3000/settings`
   - 查看积分余额
   - 查看交易记录
   - 兑换码兑换

## ⚠️ 注意事项

1. **管理员权限**：只有管理员账号才能访问 `/admin/users`、`/admin/redeem-codes`、`/admin/credits` 等页面
2. **普通用户**：如果使用普通用户账号登录，访问管理员页面会被重定向到 `/admin`
3. **账号配置**：账号信息在 `.env.local` 文件中的 `ADMIN_USERS` 环境变量中配置

## 🆘 如果无法登录

1. 检查 `.env.local` 文件中的 `ADMIN_USERS` 配置
2. 确保后端服务正在运行（`http://localhost:3001`）
3. 检查浏览器控制台是否有错误信息
